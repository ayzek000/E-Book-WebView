Для создания иконок PWA вы можете использовать следующие инструменты:

1. PWA Builder: https://www.pwabuilder.com/
2. App Manifest Generator: https://app-manifest.firebaseapp.com/
3. Maskable.app Editor: https://maskable.app/editor

Вам нужно создать иконки следующих размеров:
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png
- maskable-icon.png (196x196 с дополнительным пространством вокруг логотипа)

Рекомендуемый цвет фона для иконок: #4F46E5 (индиго)
